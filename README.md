# Escape-CADT
Escape CADT, 2D puzzle adventure game built entirely in C using the Raylib graphics framework. Trapped late at night inside the campus complex, players explore the hallways, bypass security restrictions using item access keys, and outsmart a rogue AI companion named C.A.T by solving a randomized puzzles.

Escape CADT
Language: C

Graphics Library: Raylib 6.0

To compile:
gcc Code.c -o game.exe -lraylib -lopengl32 -lgdi32 -lwinmm
To run:
./game.exe

The file besttime.txt is used to store the best completion time.
